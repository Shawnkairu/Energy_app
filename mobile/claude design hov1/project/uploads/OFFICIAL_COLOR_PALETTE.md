# Official Color Palette and Character

This image is the official e.mappa color and character reference.

![e.mappa official color palette and character](./ui-references/emappa-official-color-palette-character.png)

Use this character as the emotional source for e.mappa's product surfaces: warm, quiet, trustworthy, soft, precise, and human. The app should feel like beautiful warm artifacts laid onto a clean canvas, not a generic dashboard.

## Usage Rules

- Prefer white and warm cream as the main app canvas.
- Use `fur_cream`, `plush_caramel`, and `scarf_oat` for elevated surfaces, cards, and soft panels.
- Use `fox_orange`, `rust_brown`, and `guitar_maple` for primary actions, highlights, and warm infographics.
- Use `espresso_shadow`, `burnt_chestnut`, and `near_black_brown` for text, linework, and deepest detail.
- Avoid stray blues, purples, and cold grays unless a status state genuinely needs contrast.
- Keep the character feeling: soft, minimal, warm, trustworthy, and slightly playful without becoming childish.

The machine-readable extraction is stored at `docs/ui-references/emappa-official-color-palette.json`.
